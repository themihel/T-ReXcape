# T-ReXcape
T-ReXcape is a small local multiplayer for 2 persons made during a school project. All graphics are hand-drawn to achieve a paper-like effect.

## Regulations
- C# / Windows Forms
- Local, turn-based multiplayer
- Mouse control
- No DLLs
